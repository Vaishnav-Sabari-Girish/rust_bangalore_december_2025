# Rust Bangalore December Meetup 

This is my presentation for the Rust Bangalore December Meetup held on 20/12/2025. 

## View the slides

To view the slides you can use the `present.sh` script.

```bash
./present.sh
```

Or directly via the `presenterm` command as follows

```bash
presenterm ./presentation.md -x -c ./config/presenterm.yml -p
```

## Export the slides

You can also export the slides to PDF

Use the `export.sh` Script for that.
